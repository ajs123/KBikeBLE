 - -- Calibration ---------------------------------- 
showcal - Show the calibration currently in use. 
factor - <float> Enter a new cal factor (should not be necessary). 
offset - <float> Enter a new calibration offset (follow with activate). 
calibrate - Start tool-based calibration. Requires the Keiser cal tool. 
activate - Make the new calibration values active. Requires confirmation. 
write - Write currently active calibration to the calibration file. Requires confirmation. 
read - Read calibration from the calibration file. 
defaults - Set calibration to hard-coded defaults. Requires confirmation. 

 - -- Utility and debugging ------------------------- 
batt - Show battery status. 
res - <optional int> Monitor ADC readings - indicated number, or any input to stop. 
adcref - Toggle the analog reference between default 3.6V and Vdd. 
adccal - Calibrate the ADC offset. 
temp - Show the chip temperature (relevant to ADC offset) 
c - Enter a comment (shows in the terminal log.
